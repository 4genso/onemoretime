ほーいっ☆
今日はエネルギー温存(←わら)と、 滋賀県・富山県調べ に時間を費やしました(^^)/
 
 
 
 
エネルギーは２日間で有り余るほどいただきましたよもちろん。
だけど、２日間てんしょん上がりすぎたみたいなので
クールダウンも必要かな、と思いまして('・ω・)。
 
 
 
 
もちろん下調べは徹底する人です。
何せ、遠足の３日前には荷物も着る服も準備完了にしておくようなイベント好きだからね( // ω // )
 
 
 
 
...とゆーて、まだまだ滋賀県と富山県のこと知りたいので
情報提供待ってます
 
 
 
 
あ。ちなみに滋賀県の ひこにゃん ねー、
ひめたんとひこにゃん、誕生日一緒なんだよ!
これ1年半くらい前にはすでに知ってた(`・ω・')どや
 
ひこにゃんめっちゃ可愛いやん!!!
ひめたんとひこにゃんて名前のニュアンス近いし!
勝手な親近感っ。
 
 
 
 
うん。
りんごちゃん も ろってぃー も更新できたね☆
 
え？伝言預かる前に、２人に更新の仕方教えてあげるのが先でしょって？
 
ちっちっち。携帯によって相性があるらしいんですどうやら('・ω・`)
出来てよかった!おめでとーう♪♪
 
 
 
 
色...。
変わってるかな(ω)？
 
地元広島を始め、大阪、滋賀、富山でイベントが出来ることを
いまはどう思ってますか？
 
私たちのこと支えてくださるみなさんのおかげでイオンツアー参加できて
地元のみなさんに頑張るねって言えたこと、
大阪で、たくさんの方に出会えたこと本当に嬉しかったです!
アンダーだから余計に昨日今日が初めましての方もたくさんいて、
ひめたんのこと知ってもらえるいいきっかけになったかなーと(ω)うん
滋賀、富山のみなさんにも早くご挨拶したくて、今から楽しみです!!!
 
 
テストの前日に何かやってる事ってありますか？
 
ひめたんはどちらかと言えば、
テスト1週間前くらいから詰めてつめてテスト前日はいっぱい寝る!
テスト本番になって頭がぼーっとしたらある力も出せないからね？
 
 
滋賀県もお弁当美味しいよ
いつか、滋賀県にも来てくださいね!
 
超ぐっどたいみんぐ!
明日滋賀県行くんですよ(*^^*)
滋賀県でも美味しいお弁当食べたいなー♪
 
 
どうやったら真剣に勉強できますかね？
 
短い時間でいいから集中するんです(ω)
15分お勉強頑張ったら30分テレビ観ちゃるー!って。
あとはテスト後のご褒美を考えるとか☆☆
...てかテスト週間なのにコメントしてくれてありがとう∩^ω^∩★
本番も頑張ってね!いい報告待ってますっ
 
 
大阪テンション高いねｗｗ富山はそんなにテンション高くないかも!?
 
いやいやそんなことっ!
初富山。そりゃもう始まる前からうきうきうずうずしてるよ(o>ω<o)♪
みなさん一緒に盛り上げてくださったら嬉しいなー。
