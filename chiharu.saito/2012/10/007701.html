斎藤ちはる OFFICIAL BLOG
2012/10 10 Wed 斎藤ちはる アイドルって、なんていい職業なんだろうか。
 
こんにちはるん(`･ω･´)
 
 
前回は、本当にたくさんのコメント、ありがとう(/ _ ; )
 
 
 
応援してくださっている方がいることがいんなにいるってことに今回でやっとわかって、
なんてばかだったんだろう、
もっと早く気づけよう、
と少し後悔してしまいました。
 
 
 
初めて、220を超えるコメント数がきて、ほんとう驚きました。
 
 
こんなに応援してくださっている方がいるんだ...！！！
 
見てくれている人がいるんだ...！！！
 
感動しました(/ _ ; )
 
 
 
 
そして、少し自信にもつながりました。
 
 
見てくれている人がいる。
 
応援してくれている人がいる。
 
 
それだけで自信に繋がるって
それだけ皆さんの力が凄いってことだと思うんです。
 
 
 
 
 
 
自信が持てない人だってたくさんいるんだから無理に自信をつけなくたって、焦らなくたって、大丈夫だよ。
 
 
 
ってコメントしてくださる方もいました。
 
 
 
ありがとう(/ _ ; )
 
 
 
 
でも、人前に立つ職業。
 
 
 
 
自信がないのに歌ったり踊ったり喋ったりすることって
すごい迷惑だし非常識なんじゃないかなって、思った。
 
 
 
 
みなさんが気付かせてくれた。
 
 
 
ありがとう(/ _ ; )
 
 
 
 
応援してくれて、
励ましてくれて、
ときには厳しい意見をくださる。
 
 
 
そうやって色んな意見を下さった皆さんに、
元気な笑顔でお返しをする。
 
 
 
アイドルって、なんていい職業なんだろう！
 
 
 
 
 
 
アイドルって、思っていたよりすごく辛いこともいっぱいあるよ。
怖くなることもあるよ。
 
 
 
だけとね、でもね、ほんとにいっぱい、嬉しいこと、楽しいこと、幸せなこと、いっぱいある！
 
 
この職業に出会えてよかった。・゜・(ノД`)・゜・。
 
 
 
 
私は顔的にも身長的にも性格的にもアイドルっていうよりかは
女優さんとかモデルさん方向だと思うからアイドルって難しいけど、
 
人を笑顔にしたいって気持ちだけは負けないからさ。
 
 
 
アイドルを全うします。
全力です。
 
 
 
夢を与えるお仕事。
笑顔を届けるお仕事。
幸せを配るお仕事。
 
 
 
それを全部ひっくるめたものがアイドルだとわたしは思う。
 
 
 
 
向いてるか向いてないかは関係ない！
 
 
アイドルってゆー、そのお仕事、職業が大好きな気持ちが大事だと思う！
 
 
 
わたしは、アイドルが、大好きです。
 
 
 
 
夢を追いかけたいんです。
 
前を目指したいんです。
 
 
 
 
だから、負けない。
 
自信を持って、頑張りたい。
 
頑張る。
 
 
 
 
 
今回のブログは、いつも通り元気なブログを書こうかと思ったけど、
 
コメントで皆さんが真剣に書いてくださったのを見て、
 
わたしも真剣に、自分の思いを伝えよう。
 
皆さんのコメントを見て考えたことを伝えようって思った。
 
 
 
 
いつもいつも、
読んでくれてありがとう(/ _ ; )
 
 
ほんとうにありがとう(/ _ ; )
 
 
 
 
皆さんが、一番だよ！！
って言ってくれる笑顔を大切に、忘れずに頑張りますね！！！
 
 
笑顔が、伝染していけばいいな♪♪！
 
 
 
 
こんな感じで、握手、待ってますよ！
 
 
いやーやっぱちょいまち！
 
握手会では、これ以上の笑顔、期待しててください(o^^o)
 
 
 
 
 
(p`･ω･´q)
 
 
ちはるん。
