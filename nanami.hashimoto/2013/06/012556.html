橋本奈々未 OFFICIAL BLOG
2013/06 22 Sat 橋本奈々未 (´_ゝ｀) BADBOYSJ 最終回
&nbsp;
&nbsp;
こんばんは！ななみです！
&nbsp;
&nbsp;
今日はBADBOYSJ、遂に最終回です！！
&nbsp;
&nbsp;
ほんとあっという間。
４月から放送が始まり、もう３ヶ月経つんですね...早かったぁ
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
乃木坂ではなく、外の世界で
たくさんの方たちと１つの大きなものを作るっていう作業は
私にとってBADBOYSJが初めてでした。
&nbsp;
&nbsp;
役をもらっての演技のお仕事も初めてでした。
&nbsp;
&nbsp;
&nbsp;
最初は本当に自分のことで精一杯で...
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
でも同年代のキャストさんたちの
仕事に対する熱意だったり
プロ意識の高さだったり
もちろん、演技だったり
&nbsp;
狭い世界しか知らなかった私にとって、１つ１つが本当に刺激的で勉強になりました。
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
オンエアを観て自分に対して悔しすぎて泣いたこともあったし
&nbsp;
演技をほめられたときは
心から嬉しかったし
&nbsp;
毎週たくさんの人にこの作品を観て欲しいって思った
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
その中で映画の撮影も始まって、
ドラマの反省とプリンシパルでの経験を久美に生かそうと臨みました。
&nbsp;
&nbsp;
どうだろう、出来上がりを観るのがドキドキです。笑
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
モノをつくるってこういうことなのかって知りました
&nbsp;
たくさんの人がそれぞれ全力で１つの作品に向かって
みんなの努力や苦労を感じながら自分も精一杯頑張って
そうしていくうちにその作品が大好きになっていくんだなって思いました
&nbsp;
みんなが真剣であればあるほど
その作品って愛されるんだろうなって思いました
&nbsp;
&nbsp;
&nbsp;
&nbsp;
私はこのBADBOYSJがだいすきです
皆さんにもだいすきになって欲しいです
 
&nbsp;
初めてのことがたくさん詰まったこの作品は、私にとってきっとこの先も大事なものになると思います。
&nbsp;
久美という役にさようならするのは寂しいけど、
皆さんに一時でも私を久美として記憶されることがあれば嬉しいなと思います。
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
そんなBADBOYSJ 最終回
今日深夜00:50〜日テレ で放送になります！！
&nbsp;
是非、リアルタイムで観て欲しいです！
私もリアルタイムで観ます！！
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
最終回はね、泣きました
佐々木さんと司くんの戦い
石本さんの思い
その後...
皆さん必見です！
&nbsp;
&nbsp;
よろしくお願いします(*^_^*)
 
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
&nbsp;
ではまた、夜に♪
